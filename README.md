# Hotel Booking — End-to-End QA Project

A complete quality-assurance project for a hotel booking application, covering
**manual testing, API testing, SQL data validation, and UI/API automation**.

**Application Under Test (AUT):** [automationintesting.online](https://automationintesting.online)
— the Restful Booker Platform (a real hotel booking web app + REST API built for test practice).

---

## What this project demonstrates

| Area | Tools |
|------|-------|
| Manual testing | Test plan, test cases, RTM, bug reports |
| API testing | Postman |
| Database validation | MySQL, SQL |
| API automation | Python, Requests, Pytest |
| UI automation | Selenium, Page Object Model |
| Reporting | pytest-html, logging |
| Version control | Git, GitHub |

## Repository structure

```
hotel-booking-qa/
├── docs/          # Requirements, test plan, test cases, RTM, bug reports
├── api_tests/     # Pytest + Requests API automation   (Sprint 5)
├── db_validation/ # MySQL schema + SQL validation       (Sprint 4)
├── ui_tests/      # Selenium Page Object Model tests     (Sprint 6)
├── postman/       # Postman collection + environment     (Sprint 3)
├── reports/       # Generated HTML test reports
└── logs/          # Generated run logs
```
